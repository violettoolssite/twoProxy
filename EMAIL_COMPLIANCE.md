# 临时邮箱服务合规和安全指南

## ⚠️ 重要法律风险提示

临时邮箱服务可能被动接收违法内容（如儿童色情、恐怖主义等），这些内容会留在服务器上，运营者可能面临监管调查。

## 🛡️ 已实施的合规措施

### 1. 内容过滤机制

- ✅ **自动检测可疑内容**
  - 检测可疑附件和链接
  - 检测加密或编码内容（可能是逃避检测）
  - 检测可疑发件人域名（如 .onion, .i2p）

- ✅ **自动阻止和删除**
  - 检测到可疑内容时，不存储邮件
  - 只记录元数据（发件人、收件人、时间、风险等级）
  - 不存储实际邮件内容

### 2. 日志记录

- ✅ **可疑活动日志**
  - 记录所有被阻止的邮件元数据
  - 保留30天用于合规审查
  - 包含风险等级和检测模式

- ✅ **告警机制**
  - 检测到高风险内容时自动告警
  - 可以集成到监控系统

### 3. 数据保留策略

- ✅ **短期保留**
  - 正常邮件：24小时自动删除
  - 可疑活动日志：30天保留

- ✅ **最小化存储**
  - 限制邮件内容大小
  - 不存储附件
  - 可疑邮件不存储内容

## 📋 用户协议和免责声明

### 用户使用条款

1. **禁止用途**
   - 禁止用于接收或传播违法内容
   - 禁止用于儿童色情、恐怖主义等违法活动
   - 禁止用于垃圾邮件、钓鱼邮件等

2. **用户责任**
   - 用户对其使用邮箱的行为负责
   - 用户需遵守当地法律法规
   - 用户需配合监管调查

3. **服务限制**
   - 服务可能随时终止
   - 可疑内容会被自动删除
   - 不保证邮件送达

### 免责声明

1. **被动接收**
   - 本服务为被动接收邮件，不主动发送
   - 无法控制发件人发送的内容
   - 对接收到的内容不承担责任

2. **内容过滤**
   - 已实施内容过滤机制
   - 但无法保证100%过滤所有违法内容
   - 建议用户谨慎使用

3. **监管配合**
   - 如收到监管要求，将积极配合调查
   - 提供可疑活动日志
   - 协助执法部门调查

## 🔧 技术实现

### Cloudflare Worker 配置

1. **内容过滤 Worker**
   - 文件：`cloudflare-email-worker-content-filter.js`
   - 功能：检测和阻止可疑内容
   - 日志：记录可疑活动

2. **KV 存储**
   - `EMAILS_KV`：存储正常邮件（24小时过期）
   - `SUSPICIOUS_LOGS_KV`：存储可疑活动日志（30天过期）

3. **环境变量**
   - `CONTENT_FILTER_ENABLED=true`：启用内容过滤

### 部署步骤

1. **更新 Worker**
   ```bash
   # 在 Cloudflare Dashboard 中
   # 1. 进入 Workers & Pages
   # 2. 选择邮箱 Worker
   # 3. 替换代码为 cloudflare-email-worker-content-filter.js
   # 4. 绑定 SUSPICIOUS_LOGS_KV 命名空间
   ```

2. **创建 KV 命名空间**
   ```bash
   # 在 Cloudflare Dashboard 中
   # 1. 进入 Workers & Pages > KV
   # 2. 创建新命名空间：SUSPICIOUS_LOGS_KV
   # 3. 绑定到 Worker
   ```

3. **配置告警**
   - 可以集成 Cloudflare Workers 的 Webhook
   - 或发送邮件通知到管理员
