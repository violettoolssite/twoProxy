Docker 镜像加速

镜像搜索与加速拉取

Docker
搜索结果会同时给出香港 / 广州两个节点的加速拉取命令，并注明节点信息，点击复制即可使用。

推荐通过 Docker Daemon 配置永久使用镜像加速（同时配置香港 / 广州两个节点，Docker 会自动选择可用节点）：

cat <<'EOF' | sudo tee /etc/docker/daemon.json
{
  "registry-mirrors": [
    "https://mirror.yljdteam.com",
    "https://violetteam.cloud"
  ]
}
EOF
sudo systemctl restart docker

上述配置会把香港（mirror.yljdteam.com）和广州（violetteam.cloud）都作为 Docker 镜像仓库加速节点，无需再为单个镜像手动改命令。
